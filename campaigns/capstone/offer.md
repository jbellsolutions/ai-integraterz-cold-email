# Capstone Offer

## The deal (1-line)

Free expert AI implementation on a real workflow problem, run 6-10 weeks by an AICO candidate under Justin's Microsoft-certified oversight, in exchange for a real project to certify on. No charge. No upsell pressure.

## What the host business gets

| Component | Detail |
|---|---|
| Free expert AI implementation | On a real workflow problem the host scopes during intake |
| Microsoft-certified oversight | Justin (AI CO / AI CIO / AI CTO) is in the background every week |
| Working solution they keep | At the end of the project, host owns whatever was built |
| No-obligation after-conversation | If they like the work and want more, we talk. If not, no sales follow-up. |
| Pre-built role kit deployment | Candidate doesn't start from zero — kit per industry already exists |

## What the host commits to

| Commitment | Detail |
|---|---|
| 1-hour intake conversation | So the candidate can scope the project properly |
| Weekly 30-min check-in | For the duration of the project |
| Honest end-of-project feedback | This is what the candidate certifies on — feedback IS the deliverable for them |
| Patience | 6-10 weeks, not 30 days. This is a learning context (with quality controls) |

## Pricing context

**No charge to the host.** The Capstone is the cost of running the certification process well. AI Integraterz earns nothing on the project itself.

If the host asks during the intake whether they can hire the candidate or buy more work after, the answer is "yes, that's a separate conversation we can have when the project ends" — but that's never raised by us in the cold email.

## The honest after-relationship line

If the email has room for one closing-context line, this is the one:

> "If you like the work and want more help, we can help you with that. That's not the goal of the Capstone. The goal is the project, the candidate's certification, and you getting your problem solved."

## Who this offer is NOT for

- Pre-revenue / zero employees / no real workflow
- Hosts already deeply engaged with another agency on the same problem
- Problems too vague to scope in a 1-hour intake
- Hosts visibly looking for a sales call, not free help (still send if other signals are strong; mismatch caught on intake)
- Industries the candidates have not been trained for (default to AI Integraterz niche map)

## CTA shape

Low-friction reply or a short scoping call. The agent decides which:
- "Reply with the workflow you'd want help on" — lowest friction
- "15-min scoping call" — for a heavier intake
- "Share the role you're hiring for and I'll see if it fits" — when an active job posting is the trigger

None of these should sound like a sales pitch.
