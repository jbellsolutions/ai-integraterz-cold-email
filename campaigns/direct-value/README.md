# Direct Value (paid AICO offers)

This angle covers the **three paid AI Integraterz offers**. Strategy squad picks which sub-offer per prospect based on the lead segment:

| Sub-offer | Audience | File |
|---|---|---|
| **AICO** (flagship) | SMB 1-30 employees ready to install an AI-native OS across the business | [offers/aico.md](offers/aico.md) |
| **Expert Series** | Solopreneur experts, coaches, course creators, newsletter operators, podcasters | [offers/expert-series.md](offers/expert-series.md) |
| **Sovereign Blueprint** | Sales-driven SMBs — recruiters, real estate, mortgage, insurance, marketing agencies, auto dealers | [offers/sovereign-blueprint.md](offers/sovereign-blueprint.md) |

All three are AICO placements. They differ by **who** the AICO is placed with and **what** the AICO is placed to do.

The audit call is the universal CTA across all three: https://aiintegraterz.com/audit (Free Business Integration Audit, 60-90 min, run personally by Justin).

## When Strategy squad picks this angle (vs Capstone)

- Lead has revenue + headcount that fits one of the three offers AND has shown buying intent or scaled past the "free help" stage. Capstone is for hosts willing to participate in a learning context; direct-value is for hosts who want a paid, faster, definitive engagement (30-day sprint, paid placement).
- If unsure, default to **Capstone** — lower friction, no ask of money, builds the relationship.
